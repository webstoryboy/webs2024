1부터 100까지 출력하기(초기값0)

for()문을 이용하여 1부터 100까지 출력하는 예제입니다.

{
        for(let i=0; i<10; i++){
                console.log(i);
        }    
}

1줄 : for(let i=0; i<10; i++): 이 부분은 "for" 루프의 시작을 나타냅니다. 루프는 세 가지 세그먼트로 구성됩니다.
1줄 : let i=0: 반복문을 제어하는 변수 i를 0으로 초기화합니다.
2줄 : i<10: 조건을 검사하여 루프가 실행될지 결정합니다. i가 10보다 작을 때만 루프가 계속 실행됩니다.
3줄 : i++: 루프의 각 반복이 끝날 때마다 변수 i를 1씩 증가시킵니다.
4줄 : {: 이 부분은 "for" 루프의 본문을 시작하는 중괄호를 나타냅니다.
5줄 : console.log(i);: 이 부분은 현재 반복 횟수를 나타내는 변수 i의 값을 출력하는 코드입니다. "console.log" 함수는 브라우저나 Node.js 환경에서 콘솔에 값을 출력하는 함수입니다. 6줄 : }: 이 부분은 "for" 루프의 본문을 마치는 중괄호를 나타냅니다.
